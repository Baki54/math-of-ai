## পরিচিতি

এটি এমন একটি প্রজেক্ট যেখানে Machine Learning অ্যালগরিদমগুলো কোনো Machine Learning Library ব্যবহার না করে শুধুমাত্র Python এবং গণিতের সাহায্যে বাস্তবায়ন করা হবে।

## উদ্দেশ্য

এই প্রজেক্টের লক্ষ্য হলো Machine Learning-কে একটি Library হিসেবে নয়, বরং একটি গাণিতিক ধারণা হিসেবে বোঝা। প্রতিটি অ্যালগরিদমের পেছনের গণিত, যুক্তি এবং কার্যপ্রণালী ধাপে ধাপে বিশ্লেষণ করা হবে এবং পরে Python দিয়ে বাস্তবায়ন করা হবে।

# Machine Learning from Scratch

---

A collection of Machine Learning algorithms implemented from scratch using pure Python and mathematics, without relying on Machine Learning libraries.

## About

This project focuses on understanding how Machine Learning algorithms work internally rather than simply using them through libraries. Every algorithm will be built step by step from its mathematical foundations and then implemented in Python.

## Topics

* Linear Regression
* Logistic Regression
* Decision Trees
* Random Forests
* Neural Networks
* Transformers
* Large Language Models
